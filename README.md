# MCS3

ubuntu上使用的命令行proxy设置小工具

- 无需GUI的交互
- 常用功能集成
- 支持多种后端
- 支持 amd64,arm64,armv7
- 自带精简的规则

## 常用命令

```
myclash #查看当前myclcash状态
myclash service update_subscribe # 更新订阅
myclash change_subscribe <订阅名>  # 手动切换订阅(不下载新文件)

myclash shell on 
myclash shell off

myclash window off
myclash window on

myclash ui 

```
